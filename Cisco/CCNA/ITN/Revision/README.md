# ITN - Introduction to Networks
## Révisions Cisco CCNA - Cours 1/3

---

## 📚 Description du cours

**Introduction to Networks (ITN)** est le premier des trois cours de formation Cisco NetAcad pour la certification CCNA. Ce cours couvre les fondamentaux des réseaux informatiques.

**Durée estimée :** ~70 heures  
**Modules :** 17 modules + examens

---

## 🎯 Objectifs d'apprentissage

Ce cours couvre :
- Les concepts fondamentaux des réseaux
- Modèles OSI et TCP/IP
- Couches physiques et liaison de données
- Protocoles réseau (Ethernet, IP, TCP, UDP)
- Adressage IPv4 et IPv6
- Configuration de base des équipements Cisco
- Sécurité réseau fondamentale
- Conception de petits réseaux

## 💡 Méthodologie d'étude

### Pour chaque module :
1. **Visionner** le cours NetAcad + vidéos Jeremy's IT Lab
2. **Prendre des notes** dans `Notes/Module-XX-notes.md`
3. **Créer la fiche** de synthèse dans `Fiches/`
4. **Faire le QCM** du module pour valider les acquis
5. **Réaliser les labs** Packet Tracer

### Pour chaque Checkpoint :
1. **Réviser** les fiches des modules concernés
2. **Refaire** les QCM des modules
3. **Passer le QCM Checkpoint** (conditions d'examen)
4. **Analyser** les erreurs et revoir les points faibles

### Préparation finale :
1. **Réviser** toutes les fiches (1-17)
2. **Refaire** tous les QCM
3. **Passer** les examens blancs
4. **Labs** Packet Tracer récapitulatifs
5. **Anki** : révisions espacées quotidiennes

## ⚡ Commandes IOS essentielles

### Modes de commande
```cisco
Router> enable
Router# configure terminal
Router(config)# interface gigabitEthernet 0/0
Router(config-if)# exit
Router(config)# end
```

### Configuration de base
```cisco
hostname R1
no ip domain-lookup
enable secret class
line console 0
 password cisco
 login
line vty 0 4
 password cisco
 login
service password-encryption
banner motd # Unauthorized access prohibited #
```

### Sauvegarde
```cisco
copy running-config startup-config
show running-config
show startup-config
```
