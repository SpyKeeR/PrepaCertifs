# CCNA 200-301 - Preparation Certification
## Cisco Certified Network Associate

---

## 🎯 Objectif

Préparation complète pour l'examen de certification **CCNA 200-301**.

**Certification :** Cisco Certified Network Associate  
**Code examen :** 200-301  
**Durée :** 120 minutes  
**Questions :** ~120 questions (QCM, simulations, drag-and-drop)  
**Score de réussite :** 825/1000  
**Validité :** 3 ans

---

## 📚 Structure de formation

La préparation est basée sur les **3 cours officiels Cisco NetAcad** :

### 1. ITN - Introduction to Networks
- **Modules :** 17 modules
- **Durée :** ~70 heures
- **Contenu :** Fondamentaux des réseaux, modèles OSI/TCP-IP, adressage IP, configuration de base
- **Status :** Modules 1-7 complétés ✅

### 2. SRWE - Switching, Routing, and Wireless Essentials
- **Modules :** 16 modules
- **Durée :** ~70 heures
- **Contenu :** VLANs, STP, routage, DHCP, sécurité L2, WLAN
- **Status :** À commencer ⬜

### 3. ENSA - Enterprise Networking, Security, and Automation
- **Modules :** 14 modules
- **Durée :** ~70 heures
- **Contenu :** OSPF, ACL, NAT, WAN, QoS, automatisation
- **Status :** À commencer ⬜

## 💡 Méthodologie d'étude recommandée

### Pour chaque module :
1. **Étudier** le cours NetAcad + vidéos Jeremy's IT Lab
2. **Prendre des notes** détaillées (fichier Notes/)
3. **Créer la fiche** de synthèse (fichier Fiches/)
4. **Faire le QCM** du module pour valider
5. **Pratiquer** avec Packet Tracer

### Pour chaque Checkpoint :
1. **Réviser** les fiches des modules concernés
2. **Refaire** les QCM individuels
3. **Passer le QCM Checkpoint** en conditions d'examen
4. **Analyser** les erreurs et revoir les points faibles

### Préparation finale :
1. **Révision complète** de toutes les fiches
2. **QCM** : Refaire tous les QCM (score cible : 90%+)
3. **Examens blancs** : Conditions réelles (temps limité)
4. **Labs Packet Tracer** : Scénarios complets
5. **Flash cards Anki** : Révisions espacées

## 🎓 Compétences CCNA couvertes

### 1.0 Network Fundamentals (20%)
- Modèles OSI et TCP/IP
- Câblage et topologies
- Adressage IPv4 et IPv6
- Opérations sans fil

### 2.0 Network Access (20%)
- VLANs et trunking
- EtherChannel
- Spanning Tree
- Configuration de switchport

### 3.0 IP Connectivity (25%)
- Routage statique et dynamique
- OSPFv2 (single-area)
- Premier saut redondant

### 4.0 IP Services (10%)
- NAT/PAT
- DHCP et DNS
- NTP
- SNMP, Syslog

### 5.0 Security Fundamentals (15%)
- ACL
- Sécurité des ports
- Sécurité sans fil (WPA2/WPA3)
- VPN

### 6.0 Automation and Programmability (10%)
- APIs REST
- Automatisation (Ansible, Puppet)
- JSON, YAML
- Contrôleurs (DNA Center)
